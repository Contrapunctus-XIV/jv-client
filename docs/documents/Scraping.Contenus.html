Publications de la rédaction JVC

Je regroupe dans cette section les articles, actualités et vidéos publiés par l'équipe de rédaction. Ces formes différentes de contenus sont désignées par l'appellation catégories de contenus.

  • Actualités (ou news, ID 50) : comme leur nom l'indique, ce sont des contenus relatant des événements et informations récentes.
  • Articles (ID 51) : ce sont des contenus portant sur des thématiques en dehors de l'actualité (jeux, guides, etc.).
  • Vidéos (ID 13) : chroniques, reportages, gameplays, etc.

Commentaires

Les utilisateurs de JVC ont la possibilité de poster des commentaires sous les contenus de la rédaction. Il suffira d'utiliser la classe Content pour les récupérer étant donné un contenu.

Méthodes

Les méthodes suivies d'une astérisque sont des page scrapers.

Classe statique V4

  • V4.getContents* renvoie les contenus de la rédaction
  • V4.getHighTechContents* renvoie les contenus high-tech
  • V4.getTrendingContents* renvoie les contenus tendance
  • V4.searchArticles* renvoie les articles correspondant aux termes de recherche
  • V4.searchNews* renvoie les actualités correspondant aux termes de recherche
  • V4.searchVideos* renvoie les vidéos correspondant aux termes de recherche
  • V4.searchWikis* renvoie les wikis correspondant aux termes de recherche

Classe Content

Cette classe représente un contenu de la rédaction. Le constructeur prend en argument l'ID du contenu.

  • Content.getInfos renvoie les informations du contenu
  • Content.getComments* renvoie les commentaires des utilisateurs postés sous le contenu
  • Content.getRelatedNews* renvoie les actualités associées au contenu
  • Content.getRelatedVideos* renvoie les vidéos associées au contenu
  • Content.getRelatedWikis* renvoie les wikis associés au contenu

Classe Video

Cette classe hérite de Content et contient les mêmes méthodes que celles décrites juste au-dessus.

Classe ContentComment

Cette classe représente un commentaire d'utilisateur posté sous un contenu de la rédaction. Le constructeur prend en arguments l'ID du commentaire et le contenu parent (ID ou instance de Content).

  • ContentComment.getInfos renvoie les informations du commentaire
  • ContentComment.getAnswers renvoie les réponses postées sous le commentaire

Exemple

Pour afficher toutes les news jeu concernant la Paris Games Week :

for await (const page of V4.getContents({ query: { events: [6840], types: [54] } })) {
    console.log(page);
}
