V4Client

La classe V4Client permet de poster et de gérer des reviews (sous des jeux vidéo) et des commentaires (sous les contenus de la rédaction). Son constructeur prend en entrée une instance connectée de la classe Client.

Les fonctions fournies sont les suivantes :

  • V4Client.addComment pour ajouter un commentaire sous un contenu
  • V4Client.deleteComment pour supprimer un commentaire
  • V4Client.updateComment pour mettre à jour un commentaire
  • V4Client.addReview pour ajouter un avis sous un jeu vidéo
  • V4Client.deleteReview pour supprimer un avis

Profile

La classe Profile permet de récupérer et de modifier des informations sur son profil public. Son constructeur prend en entrée une instance connectée de la classe Client.

  • Profile.getFavorites pour obtenir les jeux, forums et topics favoris
  • Profile.editFavoriteForums, Profile.editFavoriteTopics, Profile.editFavoriteGames pour les modifier
  • Profile.setAvatar pour changer son avatar
  • Profile.setDescription pour changer son description
  • Profile.getForumPosts pour obtenir les posts du compte
Warning

Cette méthode utilise les pages de profil publiques de JVC qui sont sujettes à une limite de requêtes se situant à environ 3 par seconde. Il est donc recommandé d'utiliser une alternative dans le cas où un grand nombre de requêtes doivent être effectuées.
